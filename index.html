Projects

zkGrid

A puzzle game where players prove in zero-knowledge (via zk-SNARKs) that they found the empty cell in a dotted grid, without revealing its location. The proof is generated using Groth16, and is displayed as a QR code. The next version will integrate proof verification.

Get the code!

Repartee

A simple word association game deployed as a dapp, where you guess a word that matches a given prompt and see how many others think like you. Play to find common connections, or add new word pairs to expand the game and challenge others!
